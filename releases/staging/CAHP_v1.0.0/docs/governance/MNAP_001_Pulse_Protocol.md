## 1. Executive Summary
This proposal formalizes the **metabolic heartbeat** of the Helix lattice by establishing a daily SATS allocation tied to successful Morning Ritual completion. It transforms our constitutional ceremony into a living economic system where **trust becomes tangible energy flow**.

## 2. Constitutional Alignment
This proposal operationalizes three core principles:
1.  **Proofs, Not Promises**: SATS allocation requires verifiable "SYSTEM GREEN" status.
2.  **Grammar as Alignment**: Automated, rule-based distribution removes human friction.
3.  **Structure as Teacher**: Economic incentives reinforce healthy operational patterns.

## 3. Ratified Parameters
*   **3.1 Base Metabolic Rate**
    *   **Daily Allocation:** **365 SATS** per verified node per day.
    *   **Distribution Time:** 12:00 UTC daily.

*   **3.2 Pulse Validator Criteria (Hybrid Model)**
    *   **Role Requirement:** `Color Artist` (Level 2) or higher.
    *   **Reputation Threshold:** Minimum 30-day consecutive ritual completion streak.
    *   **Technical Capability:** L0 Identity keys operational.
    *   **Rotation Schedule:** No validator serves consecutive days; daily random selection.

*   **3.3 Validation Threshold**
    *   **Required:** 3 validators per pulse attestation.
    *   **Quorum:** 2/3 agreement triggers distribution.

## 5. The Quiet Morning Protocol (Supportive Failure Mode)
When a node fails its ritual/pulse:
*   **Automatic Status:** Node enters `METABOLIC_CONSERVATION` for 24 hours; allocation is withheld (paused, not penalized).
*   **Support Protocol:** Neighbor nodes initiate a wellness check, and automated recovery scripts are shared.
*   **Extended Quiet (7+ days):** Node enters `HIBERNATION`, requiring manual reactivation.
